# ListIt - to do list web extention

## Description

ListIt is a simple to do list web extention. It is a work in progress.

## Installation

```
git clone https://github.com/Dinex-dev/ListIt.git
cd ListIt
pnpm run dev
```

Now import the project in chromeium based browser
